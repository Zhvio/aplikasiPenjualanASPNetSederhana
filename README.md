# 🛒 ASP.NET App - Aplikasi Penjualan Sederhana Berbasis WEB

Aplikasi Web penjualan sederhana yang dibangun menggunakan **ASP.NET Core Blazor (Server-side)**.

## 🚀 Fitur Utama

- **Katalog Produk**: Menampilkan daftar produk.
- **Sistem Keranjang**: Menambah, menghapus, dan menghitung total harga produk secara real-time.
- **Riwayat Transaksi**: Mencatat setiap pembelian.
- **Responsive Design**: Tampilan yang responsive dan berjalan di beda perangkat.

## 🛠️ Tech Stack

- **Framework**: .NET 8/9 Blazor Web App
- **Language**: C# 12
- **UI Framework**: Bootstrap 5
- **Icons**: Bootstrap Icons
- **Pattern**: Partial Classes (Code-Behind) untuk pemisahan logika dan tampilan.
